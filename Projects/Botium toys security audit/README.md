# Botium Toys Security Audit

This project contains a completed internal IT audit for Botium Toys based on the NIST Cybersecurity Framework (NIST CSF).

## Objectives
- Assess existing security controls
- Identify risks and vulnerabilities
- Evaluate compliance with:
  - PCI DSS
  - GDPR
  - SOC standards

## Key Findings
- Missing encryption controls
- No intrusion detection system (IDS)
- Weak password policies
- Lack of disaster recovery planning
- Missing least privilege implementation

## Recommendations
- Implement encryption
- Deploy IDS
- Improve password management
- Enforce access controls
- Establish backup and disaster recovery procedures
