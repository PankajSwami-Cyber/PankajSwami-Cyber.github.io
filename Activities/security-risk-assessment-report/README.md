# Security Risk Assessment Report

## Project Overview
This project analyzes a data breach at a social media organization and recommends network hardening techniques to reduce future security risks.

## Identified Vulnerabilities
- Employee password sharing
- Default database administrator password
- Lack of firewall filtering rules
- No Multi-Factor Authentication (MFA)

## Security Recommendations
1. Multi-Factor Authentication (MFA)
2. Strong Password Policies
3. Firewall Configuration and Port Filtering

## Skills Demonstrated
- Risk Assessment
- Network Security
- Network Hardening
- Access Control
- Firewall Security
- Security Documentation

## Tools & Concepts
- MFA
- Password Policies
- Firewall Rules
- Port Filtering
- Principle of Least Privilege

## Outcome
The proposed controls reduce unauthorized access risks and strengthen organizational security against future attacks.
